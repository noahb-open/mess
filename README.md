# 🚀 Blue Rocket Chat

A neat, real-time messaging application inspired by Snapchat. Built with Node.js, Express, MongoDB, and Socket.io.

## 🛠️ Features
- **Secure Authentication**: Signup with email, password, and custom username.
- **Verification System**: 6-digit email verification token before launching into the app.
- **Real-Time Mesh**: Instant chat using WebSockets.
- **Friend Network**: Fast custom friend adding system.
